When updating the project

- Check if significant features, design changes, or structural modifications have been made
- Update the README.md in trickle/notes/ if necessary to reflect:
  - New features or interactive elements
  - Changes to the color scheme or design system
  - Modified animations or effects
  - Updates to the technology stack
  - New sections or components
- Keep the README concise and focused on user-facing features
- Maintain the existing format and structure
