# Peace's 18th Birthday Website

A beautiful, interactive birthday celebration website created for Peace's 18th birthday.

## Features

- **Interactive Hero Section**: Animated welcome with floating decorative elements
- **Birthday Cake**: Clickable cake with surprise animations
- **18 Candles**: Interactive candles that can be blown out individually
- **Spotify Playlist**: Embedded music player with birthday celebration songs
- **Personal Message Page**: Special page with photo gallery, slideshow, and long message
- **Wishes Section**: Beautiful wish cards with icons
- **Friends Wishes Page**: Separate page where friends can add birthday wishes
- **Celebration Gallery**: Milestone celebration cards
- **Responsive Design**: Works perfectly on all devices

## Color Scheme

- Background: Cream (#fffbf1)
- Primary: Dark Pink & Magenta gradient
- Accents: Light pink for sections

## Animations

- Floating elements throughout the page
- Candle flickering effects
- Bounce and pulse animations
- Hover effects on interactive elements

## Interactive Elements

1. Click the birthday cake for sparkle effects
2. Click each candle to blow it out
3. Hover over cards for elevation effects
4. Smooth scrolling and transitions throughout

## Technology Stack

- React 18
- Tailwind CSS
- Lucide Icons
- Trickle Database for storing wishes, photos, and messages
- Custom CSS animations
- Multi-page structure (index.html, wishes.html, message.html)

## Pages

1. **Main Page (index.html)**: Hero, cake, candles, Spotify, wishes, gallery
2. **Message Page (message.html)**: Personal message with photos, slideshow, and long text
3. **Wishes Page (wishes.html)**: Crowdsourced birthday wishes from friends
